# 🎵 NLP Lexical Analysis — *Someone Like You* by Adele

## Overview
Applied activity from the course **Fundamentos de AI (con NN y Transformers)**.  
A complete lexical analysis pipeline built in Python, applied to a fragment of  
Adele's *Someone Like You* (2011).

## Sections covered
1. Text Description
2. Lexicons (15-entry table with category distribution chart)
3. Phonemes, Graphemes & Morphemes
4. Tokenization — sentence, word & character-level + frequency charts & word clouds
5. Rule-Based Tokenizers (regex patterns)
6. Word Normalization — lowercasing, punctuation removal, stopwords, stemming & lemmatization
6b. N-gram Analysis — unigrams, bigrams & trigrams with frequency charts & word clouds
7. Summary

## Libraries used
`nltk` · `pandas` · `matplotlib` · `wordcloud`
