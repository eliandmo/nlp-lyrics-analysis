# 🎵 NLP Lexical Analysis — *Someone Like You* by Adele

## Overview
Applied activity from the course **Fundamentos de AI (con NN y Transformers)**.  
A complete lexical analysis pipeline built in Python, applied to a fragment of Adele's *Someone Like You* (2011).

## Sections covered
1. Text Description
2. Lexicons (15-entry table)
3. Phonemes, Graphemes & Morphemes
4. Tokenization (sentence, word, character-level)
5. Rule-Based Tokenizers (regex)
6. Word Normalization (stemming & lemmatization)
7. Summary

## Libraries used
`nltk` · `pandas` · `matplotlib`
